# 🚌 BusTrack Trichy
*Real-time Bus Tracking System for Tiruchirappalli*
Built for ThinkRoot x Vortex '26 Hackathon

## 🌐 Live Demo
[View Live App](https://SharmiReddy-Kancharla.github.io/bustrack-vortex26)

---

## 📌 Problem Statement
Public bus commuters in Trichy have no real-time visibility 
into bus locations, causing uncertainty, missed buses, and 
long uninformed waits — especially on low-connectivity networks.

## 💡 Solution
BusTrack Trichy is a resilient, real-time bus tracking web app 
that works across all network conditions — from full 4G to EDGE 
to complete offline — using adaptive updates, ML-based ETA 
prediction, and GPS interpolation.

---

## ✨ Features
- 🗺️ Live map with 3 real Trichy bus routes
- 📍 Passenger live GPS location with nearest stop & walking time
- 🧠 ML ETA Prediction (Gradient Boosting Regressor)
- 📶 Adaptive network simulation (Full / 3G / EDGE / Offline)
- ⚡ Store-and-forward offline buffer
- 〰️ Path interpolation & smooth bus animation
- ⏱️ Live countdown timer to next stop
- 🔍 Search buses and stops
- 🎯 Auto-follow selected bus
- ☀️ Dark / Light mode
- 📱 Fully mobile responsive
